Second brain: solution map

Obsidian vault là nguồn sự thật; CLI ingest URL; Apify & X API ưu tiên ổn định; LLM tách evidence và suy luận; digest định kỳ; tùy chọn challenge từ digest; YouTube Phase 2+ có embed + transcript trên reader web (theo mock); Obsidian giữ vault & chi tiết.

Superpowers (brainstorming): đây là bản visualize để duyệt — cùng cấu trúc với bước “present design / incremental validation”. Nếu sơ đồ lệch với ý bạn, chỉnh design doc gốc rồi cập nhật file HTML này cho khớp.

Quyết định đã chốt

Vận hành

Solo, kích hoạt thủ công: dán URL hoặc chạy CLI khi cần.

Trending

Digest định kỳ (văn bản), mặc định gợi ý hàng tuần — cấu hình được.

Obsidian + reader web

Vault trong Obsidian là SSoT. Web xây theo mock docs/visualizations/ để đọc tổng hợp — không Quartz.

LLM

API cloud cho tóm tắt, insight và digest — tách rõ trích dẫn vs suy luận.

Thu thập

Ổn định trước: X API khi đủ quyền; Apify (actor pin version); HTTP Readability cho web đơn giản.

Độ đầy đủ

Evidence: text đầy đủ, ảnh tải về assets/, code inline hoặc snippets/.

Challenge (Phase 2+)

LLM sinh câu hỏi chỉ từ nội dung digest — kiểm tra đọc hiểu chủ đề; lưu Challenges/, tùy chọn chấm / tự đối chiếu đáp án.

YouTube + web (Phase 2+)

Transcript trong source.md; bản Việt gắn nhãn dịch LLM; timeline + mốc chú ý (click → seek). Reader web: iframe + tab EN/VI hoặc hai cột. Bản quyền: cân nhắc transcript đầy đủ — thường publish: false trong vault.
