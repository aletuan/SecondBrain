Design Notes

  • Fonts stay consistent across all themes — DM Serif Display / Literata / JetBrains Mono
  • Accent colors adapt per theme for proper contrast ratios (WCAG AA)
  • CSS variables swap — same variable names, different values per theme
  • Switcher — 3-icon toggle in top-right header, persisted to localStorage
